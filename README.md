# ventyolanapitupulu-academic-database-system-CIS-institut-teknologi-del
A relational database system designed to support the academic features of the Campus Information System (CIS) at Institut Teknologi Del. This project covers full-cycle database development: from business process analysis to physical implementation using PostgreSQL.

**📌 Project Overview**
CIS IT Del is a web-based academic information system that integrates student, lecturer, and academic administration data. This project focuses on designing and implementing the database layer that supports nine core academic business processes.

**🛠️ Tech Stack**
Database: Microsoft SQL Server
Client Tool: SQL Server Management Studio (SSMS)
Modeling Tools: ERD, CDM (Conceptual Data Model), PDM (Physical Data Model)
SQL Features: DDL, DML, Joins, Views, Aggregate Functions, Set Operators, Stored Procedures, Functions

**🗄️ Database Structure**
30+ physical tables covering all academic entities
Normalization up to 3NF (Third Normal Form)
Key entities include: Mahasiswa, Perkuliahan, KRS, Presensi, Pengumuman, Nilai, Payment, Dosen, Matkul, and more
